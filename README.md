# Deliká Bolos e Doces

Site oficial da confeitaria artesanal Deliká, localizada em Embu-Guaçu - SP.

## Estrutura do Projeto

```
/
├── index.html          # Página inicial
├── cardapio.html       # Cardápio digital
├── contato.html        # Página de contato
├── assets/
│   ├── icons/          # Ícones de redes sociais
│   └── images/         # Imagens do site
├── css/
│   ├── style.css       # Estilos principais
│   ├── _variables.css  # Variáveis CSS
│   ├── _base.css       # Estilos base
│   ├── components.css  # Componentes
│   ├── layout.css      # Layout
│   └── reset.css       # Reset CSS
└── js/
    ├── script.js       # Scripts principais
    └── store.js        # Lógica do carrinho
```

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis (Design Tokens)
- JavaScript Vanilla
- Font Awesome 6.5.0
- Google Fonts (Playfair Display + Urbanist)
